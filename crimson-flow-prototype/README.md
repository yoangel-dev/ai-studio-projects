Crimson Flow Prototype es un prototipo de aplicación generado mediante herramientas de diseño y desarrollo asistido por inteligencia artificial. El flujo inicial de la interfaz y la estructura funcional fueron creados en Stitch, donde se definieron los componentes base, la jerarquía visual y los patrones de interacción.

Posteriormente, el prototipo fue reconstruido y ampliado en Google AI Studio, utilizando agentes inteligentes para generar código fuente, vistas, estilos y lógica de navegación. Esta versión incluye:

Arquitectura inicial de la interfaz generada por IA.

Prototipado interactivo basado en flujos definidos en Stitch.

Código exportado desde Google AI Studio para desarrollo local.

Estructura modular preparada para iteraciones futuras.

Base técnica para integrar funcionalidades adicionales (estado, almacenamiento, lógica de usuario, etc.).

Este directorio documenta el estado actual del prototipo y sirve como punto de partida para su evolución hacia una aplicación completa. Su propósito es mantener trazabilidad del proceso de diseño asistido por IA y facilitar la transición desde prototipo a implementación real.
